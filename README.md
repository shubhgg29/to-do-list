# To-Do List App

## Project Overview
This is a simple To-Do app made using Flutter.  
It allows the user to add and delete tasks.

---

## Features
- Add a task  
- Show tasks in a list  
- Delete a task  

---

## Steps to Run
1. Open the project in VS Code or Android Studio  
2. Click the Run button
